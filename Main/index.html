Welcome

This page is just a demo and reference for Devs to grab asset snippets.

The core css and js assets should be loaded conditionally based on the environments.

  • Debug / Staging: should load non compressed versions.
    • style.css
    • plugins.js - Combined source of plugins (e.g. carousels)
    • src.js - Combined source of all modules (i.e. custom code)
    • main.js - Init statements
  • Production: should load minified versions
    • style.min.css
    • scripts.min.js
